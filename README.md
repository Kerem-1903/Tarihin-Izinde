# Tarihin İzinde 🏹

Orta Asya'dan Anadolu'ya uzanan Türk devletlerinin ve göç yollarının hikayesini anlatan interaktif tarih portalı.

## 📖 İçerik

- **Asya Hun Devleti** (MÖ 220 - MS 216)
- **Avrupa Hun Devleti** (375 - 469)
- **Göktürk Devleti** (552 - 745)
- **Uygur Devleti** (744 - 840)
- **Karahanlı Devleti** (840 - 1212)
- **Gazneli Devleti** (963 - 1187)
- **Büyük Selçuklu Devleti** (1037 - 1157)
- **Anadolu Selçuklu Devleti** (1077 - 1308)
- **Göç Yolları Haritası**

## 🛠️ Teknolojiler

- [Next.js 16](https://nextjs.org/) — React framework
- [TypeScript](https://www.typescriptlang.org/) — Tip güvenliği
- [Tailwind CSS 4](https://tailwindcss.com/) — Stil
- [Framer Motion](https://www.framer.com/motion/) — Animasyonlar
- [Radix UI](https://www.radix-ui.com/) — Erişilebilir UI bileşenleri
- [Lucide Icons](https://lucide.dev/) — İkonlar

## 🚀 Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini aç.

## 📦 Derleme

```bash
npm run build
npm start
```
